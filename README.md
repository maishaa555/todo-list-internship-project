# To-Do List App

A simple To-Do List web application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning Outcome
- Practiced DOM manipulation
- Improved JavaScript event handling
- Strengthened front-end development fundamentals
